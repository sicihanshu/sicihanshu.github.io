pluto与rtl sdr环境设置 与soapysdr的安装

发表于2023-11-09|更新于2023-11-14|Gnuradio-soapysdr/SDR
|字数总计:541|阅读时长:2分钟|阅读量:

soapysdr

本站所有gnuradio相关教程均使用Ubuntu20.04.6 LTS WS2版本（windows子系统），有想用子系统的可自行寻找教程安装
更新，子系统就是答辩，兼容性做的太烂了，用虚拟机了
版本为22.04

市面上的sdr都支持soapysdr协议，想要利用plutosdr或者rtlsdr进行无线电设计，需要用到一些依赖库。

soapysdr bug很多，但不用慌。

更新，用了虚拟机，没用sopay，可以跳过

其依赖libiil、libad9361、Soapysdr库
首先更新库

sudo apt-get update

1.apt安装库

sudo apt-get install libiio-utils

2.源码编译 安装与测试

sudo apt-get install libxml2 libxml2-dev bison flex libcdk5-dev cmake
sudo apt-get install libaio-dev libusb-1.0-0-dev libserialport-dev libxml2-dev libavahi-client-dev doxygen graphviz

git clone https://github.com/pcercuei/libini.git
cd libini
mkdir build && cd build && cmake ../ && make && sudo make install

git clone https://github.com/analogdevicesinc/libiio.git
cd libiio
mkdir build && cd build && cmake ../ && make && sudo make install

PATH=/usr/lib/:$PATH

iio_info -s

3.ad9361

cd ~/Download
git clone https://github.com/analogdevicesinc/libad9361-iio
cd libad9361-iio
cmake ./CMakeLists.txt
make
sudo make install

注意 没有cmake的可以自己装一下

4.SoapySDR

git clone https://github.com/pothosware/SoapySDR.git
cd SoapySDR
git checkout tags/soapy-sdr-0.7.2
kdir build && cd build
cmake ..
make -j4
sudo make install
sudo ldconfig

更新。前置安装库，有些失败问题不大

到这里，基础库就安装完了，接下来该配置pluto与rtl的环境了

sudo apt install libxml2 libxml2-dev bison flex cmake git libaio-dev libboost-all-dev
sudo apt install doxygen
sudo apt install libusb-1.0-0-dev
sudo apt install libavahi-common-dev libavahi-client-dev
sudo apt install bison flex cmake git libgmp-dev
sudo apt install swig

如果cmake遇到缺失libzstd的问题，直接sudo安装：

sudo apt-get install libzstd-dev

还有错，试试下面的：

git clone https://github.com/analogdevicesinc/gr-iio.git
cd gr-iio
cmake .
make 
sudo make install
cd ..
sudo ldconfig

如果ad9361cmake报错，问题不大。

plutosdr rtlsdr

运行如下代码：

git clone https://github.com/pothosware/SoapyPlutoSDR
cd SoapyPlutoSDR
mkdir build
cd build
cmake ..

输出一大串指令，没报错就行，rtlsdr的操作相同，将github网址中的Pluto改成RTL就行。

安装gnuradio：

sudo apt-get install gnuradio

22.04默认安装3.10版本，这个很重要，要最新版本的。
